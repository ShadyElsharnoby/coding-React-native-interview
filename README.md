## Getting Started

Follow these steps to get the project up and running on your local machine.

### Installation

yarn install
npx expo prebuild
yarn ios

### Libraries Used

-   Navigation: React Navigation
-   State Management: Zustand
-   Storage: MMKV
-   API: TanStack Query
